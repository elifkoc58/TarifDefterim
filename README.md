# 🍽️ Tarif Defterim

## Proje Hakkında

Tarif Defterim, kullanıcıların internet üzerinden yemek tariflerini görüntüleyebildiği ve favorilerine kaydedebildiği Android uygulamasıdır.

## Özellikler

* Yemek tariflerini API üzerinden çekme
* Tarif detaylarını görüntüleme
* Favorilere ekleme
* SQLite veritabanında saklama
* Kullanıcı dostu arayüz

## Kullanılan Teknolojiler

* Kotlin
* Android Studio
* Jetpack Compose
* SQLite
* TheMealDB API

## Ekranlar

1. Ana Sayfa
2. Tarif Detay Sayfası
3. Favoriler Sayfası

## API

TheMealDB API kullanılmıştır.

https://www.themealdb.com/
